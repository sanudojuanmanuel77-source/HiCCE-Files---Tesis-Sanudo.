# HiCCE-Files---Tesis-Sanudo.
El trabajo final de grado "Diseño del firmware para el HiCCE en AP SoC, tiene como objetivo lograr el streaming de datos desde la placa HiCCE-128. Esta placa puede posee 128 canales para adquirir biopotenciales. Este streaming de datos se adapta al protoco LSL , y se analisán, visualizán y registran las señales en software compatible.

En este repositorio se ponen a disposición los archivos VHDL de los modulos utilizados y sus respectivos testbench. También se encuentran los archivos de lenguaje C del PS.
